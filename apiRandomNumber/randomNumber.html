API Número Aleatorio

Este sitio web envia una petición a un API externo que devuelve un número aleatorio como respuesta, el valor depende de un valor mínimo y máximo que introduzcas en los siguientes inputs de texto.

API: https://api-random-number-miguezpz.onrender.com/randomNumber/min-value/max-value

La respuesta del servidor puede demorar más de 20 segundos en la primera petición.
